Privacy Policy

Effective Date: July 20, 2026

This Privacy Policy describes how the Willhaben AI Tools browser extension handles user data. We are committed to maintaining transparent data processing workflows directly inside your browser container.

1. Information Collection and Use

The extension does not track, collect, or upload any personally identifiable configuration logs to an external private tracking server. All processes occur purely client-side.

  • Authentication Keys: If you supply an optional Gemini API Key, it is saved strictly inside your browser instance using secure local execution frameworks (chrome.storage.local).
  • Website Content Interaction: The extension analyzes user-provided listing image elements and localized search index trees purely to facilitate form auto-completion routines.
  • User Interface Activity: The system logs click triggers on price suggestion modules solely to pass chosen values into native marketplace fields.

2. Data Transmission

To provide automated smart text outputs, the extension securely transfers user-uploaded image binaries alongside standard marketplace metadata strings to the public Google Gemini API endpoints. This data transmission is performed entirely using your independent API access credentials. No secondary network paths or developer infrastructure relays are introduced.

3. Third-Party Services

This extension interacts with third-party service infrastructures directly associated with Google Gemini API models. Please review the official Google Privacy Policy guidelines regarding active automated data retention boundaries on developer platforms.

4. Changes to This Policy

We reserve the right to modify this privacy structure at any time. Any revisions will be actively reflected via instant codebase iterations updated here.

5. Contact

If you have any questions regarding data usage configurations, feel free to reach out via our open developer profile networks.
